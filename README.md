# SpacePool is small and ease to use.

> Version V0.1Beta

> Simple API

1. Just use it. (it will auto create your pool)

```
// Spawn
YourGameObject.Spawn();

// Recycle
YourGameObject.Recycle();
```

2. Choose create pool

```
YourGameObject.CreatePool();
```
